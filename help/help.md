# MTCode — Руководство пользователя

## Обзор

MTCode — лёгкий текстовый редактор для работы с PLC-файлами и обычным кодом. Построен на Tauri + React + Monaco Editor.

## Горячие клавиши

| Клавиша | Действие |
|---|---|
| `Ctrl+O` | Открыть файл |
| `Ctrl+S` | Сохранить файл |
| `Ctrl+Shift+S` | Сохранить как… |
| `Ctrl+Shift+Alt+S` | Сохранить все |
| `Ctrl+W` | Закрыть вкладку |
| `Ctrl+Shift+W` | Закрыть все вкладки |
| `Ctrl+F` | Найти в файле |
| `Ctrl+H` | Заменить в файле |
| `Ctrl+Shift+F` | Найти в проекте |
| `Alt+↓` / `Alt+↑` | Следующий / предыдущий результат поиска |
| `Ctrl+R` | Заменить всё |
| `Ctrl+G` | Перейти к строке |
| `F2` | Переключить закладку |
| `Alt+PgDn` / `Alt+PgUp` | Следующая / предыдущая закладка |
| `F1` | Палитра команд |
| `F7` | Форматировать документ (Beautify) |
| `F8` | Закрыть текущую вкладку |
| `F9` | Завершить CNC-процессы |
| `F10` | Настройки |
| `Alt+Z` | Переключить перенос строк |
| `Alt+M` | Свернуть / развернуть блок |
| `Ctrl+,` | Настройки |
| `Ctrl+Tab` / `Ctrl+Shift+Tab` | Следующая / предыдущая вкладка |

### F-панель

Под строкой состояния расположена панель быстрых действий F1–F10. Назначение каждой клавиши настраивается в **Settings → F-Key Bar**.

## Интерфейс

### Боковая панель (File Tree)

- **N** — новый файл (untitled)
- **📁+** — создать папку в выбранной директории
- Клик по файлу открывает его во вкладке
- Клик по папке раскрывает/сворачивает содержимое
- Перетаскивание файлов не поддерживается

### Вкладки

- Поддерживается множественное открытие файлов
- Грязные вкладки (с несохранёнными изменениями) помечаются индикатором `●`
- При закрытии грязной вкладки появляется диалог: **Save & Close / Discard / Cancel**
- При выходе из приложения — диалог для всех грязных вкладок

### Панель поиска

- **Ctrl+F** — поиск в текущем файле с подсветкой результатов
- **Ctrl+Shift+F** — полнотекстовый поиск по проекту с фильтрами (расширение, регистр, regex)

### Строка состояния

- **Encoding** — текущая кодировка файла
- **Ln / Col** — позиция курсора
- **🌙 / ☀️** — переключение темы (dark ↔ light)
- **✓** (BadgeCheck) — валидация PLC-файлов (вкл/выкл)

## PLC-язык

Редактор поддерживает подсветку синтаксиса для PLC-файлов:

- **Ключевые слова**: `IF`, `ELSE`, `ENDIF`, `WHILE`, `ENDWHILE`, `OPEN`, `CLOSE`
- **Препроцессор**: `#ifdef`, `#ifndef`, `#else`, `#endif`, `#define`, `#include`
- **Комментарии**: `;` и `//` до конца строки
- **Числа**: `0xFF`, `0b1010`, десятичные
- **Строки**: `"..."`, `'...'`
- **Каналы**: `&1`, `&2`, …
- **Фолдинг**: автоматический по `IF/ENDIF`, `WHILE/ENDWHILE`, `OPEN/CLOSE`, `#ifdef/#endif`

### Радужные блоки (Rainbow Blocks)

Вложенные блоки подсвечиваются разными цветами для визуального различения уровней вложенности. Включается/выключается в настройках. Поддерживает препроцессорные директивы (`#ifdef` / `#endif`).

### Валидация PLC

При включённой валидации (кнопка ✓ в строке состояния) при форматировании (`F7`) проверяется корректность PLC-кода:

- Неспаренные `IF/ENDIF`, `WHILE/ENDWHILE`
- Синтаксические ошибки
- Ошибки отображаются как маркеры в редакторе и как toast-уведомления

## Настройки

Открываются через **Settings → Open Settings** (`Ctrl+,`):

| Параметр | Описание |
|---|---|
| **Theme** | Тема оформления (dark / light варианты) |
| **Font Size** | Размер шрифта редактора |
| **Tab Size** | Количество пробелов на табуляцию |
| **Word Wrap** | Перенос длинных строк (on / off) |
| **Autosave** | Режим автосохранения: off / focusChange / delay |
| **Autosave Delay** | Задержка автосохранения (мс) |
| **Hotkeys** | Пользовательские горячие клавиши |
| **Search Collapsed** | По умолчанию сворачивать панель поиска |
| **PLC Rainbow** | Радужная подсветка блоков |
| **Rainbow Colors** | Палитра цветов (10 слотов) |
| **F-Key Bar** | Назначение действий на F1–F10 |

## Меню

### FILE

- **Open File** — открыть файл из диалога
- **Open Folder** — открыть корневую папку проекта
- **Save / Save As / Save All** — сохранение
- **Reload** — перечитать файл с диска
- **Reload As…** — перечитать с указанной кодировкой
- **Convert To…** — конвертировать в другую кодировку
- **Rename / Delete / Move to Recycle Bin** — операции с файлом
- **Close / Close All / Close All But This** — управление вкладками
- **Exit** — выход (с проверкой несохранённых изменений)

### EDIT

- **Undo / Redo**
- **Cut / Copy / Paste**
- **Find / Replace / Find in Project**
- **Find Next / Find Previous** — навигация по результатам
- **Replace All** — заменить все совпадения
- **Go To Line…**
- **Toggle Bookmark / Next / Previous**

### VIEW

- **Toggle Sidebar** — показать/скрыть боковую панель
- **Toggle Search** — показать/скрыть панель поиска
- **Toggle Fold / Unfold All** — свёртка блоков
- **Word Wrap** — перенос строк

### SETTINGS

- **Open Settings…** — открыть модальное окно настроек

### HELP

- **Help** — это руководство (открывается во вкладке)
- **About MTCode…** — информация о версии

## Автосохранение

| Режим | Описание |
|---|---|
| **Off** | Автосохранение отключено |
| **Focus Change** | Сохраняет при переключении на другую вкладку |
| **Delay** | Сохраняет через N мс после последнего изменения |

> **Примечание:** При закрытии приложения (X-кнопка) все файлы с реальными путями автоматически сохраняются как защита от потери данных (ограничение Tauri v1).

## Поддерживаемые кодировки

UTF-8, Windows-1251 (CP1251), KOI8-R, IBM866, ISO-8859-5, ASCII, UTF-16LE

## Проектный поиск

`Ctrl+Shift+F` — поиск по всем файлам в открытой папке проекта.

**Фильтры:**
- Include / Exclude — glob-паттерны (`*.cfg`, `src/**`, `*.bak`)
- Case Sensitive / Whole Word / Regex / Invert Match
- Кодировки: UTF-8 / ANSI / ASCII / UTF-16

Результаты отображаются в панели слева с возможностью перехода к каждому совпадению.

## Палитра команд

`F1` — быстрый доступ ко всем действиям меню с поиском по названию.

## Закладки

- **F2** — поставить/снять закладку на текущей строке
- **Alt+PgDn** / **Alt+PgUp** — переход между закладками
- Закладки сохраняются для каждого файла отдельно

## Форматирование (Beautify)

`F7` — форматирование текущего документа. Для PLC-файлов при включённой валидации сначала выполняется проверка синтаксиса.

## Завершение CNC-процессов

`F9` — запуск kill.bat для завершения фоновых CNC-процессов.
